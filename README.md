# web-project
## Project 2 - HTML Foundation Page

### Theme
A travel-themed webpage highlighting favorite destinations, packing essentials,
and travel tips. The site includes a home page and a linked travel page.

### What I Learned
- How to structure a multi-page website using relative file paths
- How to use HTML elements like tables, lists, navigation bars, and images
- How inline HTML styling works before CSS is introduced
- How to validate code using the W3C Nu HTML Checker

### Challenges
- Getting the relative file paths correct between index.html and project2.html
- Understanding the difference between errors and warnings in the validator
- Learning which HTML styling attributes are considered obsolete in modern standards
